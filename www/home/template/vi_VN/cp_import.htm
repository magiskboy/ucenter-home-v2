Trang này hỗ trợ MetaWeblog API theo tiêu chuẩn XML-RPC
Địa chỉ là: {$siteurl}xmlrpc.php
Thông qua giao diện này, bạn có thể dễ dàng chuyển hàng loạt các bài blog từ các trang web khác sang trang cá nhân hiện tại;
Điều này yêu cầu trang web gốc của bạn cũng phải cung cấp giao diện API theo tiêu chuẩn XMLRPC, chi tiết vui lòng tham khảo quản trị viên của trang web đó.
